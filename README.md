This project contains automated tests written in C# using Selenium WebDriver
for the CloudQA Automation Practice Form.

## What is tested
- First Name input field
- Country input field
- State dropdown

The project follows Page Object Model (POM) structure and uses
WebDriverWait for synchronization. Tests work with the iframe present
on the page.

## Tech Stack
- C#
- Selenium WebDriver
- NUnit
- WebDriverManager
